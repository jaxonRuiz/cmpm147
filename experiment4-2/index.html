Experiment 4.2 - Infinite Tile Matching Game

Description

This is an infinite tile matching game. Match as many tiles as possible, while optionally keeping the number of misses as low as possible. Using the shift key can temporarily reveal the tiles at the cost of a point every second

Technical

Tile colors are generated at random based on a hash, which can be uncovered when selected. Matched tiles are marked as completed, and locked from selection. Though tiles are hidden, their color value for matching remains consistent due to hashing.

Reflection

I didn't expect to spend as much time as I did, but the implementation of tile selection/matching logic was an interesting challenge, especially on a hashed, infinite canvas. The game was incredibly basic initially, but incrementally adding visual elements helped to make the final result feel much more satisfying.
