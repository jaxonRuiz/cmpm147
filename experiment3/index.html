Experiment 3 - Generated Tilemap World

Description

This shows off 2 simple generated scenes, and applies generative techniques for terrains and structure creation.

Technical

Made using perlin noise, scrolling using mouse position affecting noise and offsetting building positions. Single p5 canvas was used, with an offset to show both scenes simultaneously

Reflection

This project was incredibly exciting for me, but I unfortunately didn't have enough time to achieve what I wanted. I had quite a few ambitious ideas, but have not been able to get around to them. I'm hoping to get back to this project at some point in the future though. To any graders seeing this, I may have updated it on my own since uploading the submission document.
